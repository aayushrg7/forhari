Introduction

Carch is a modular Bash script designed to streamline the initial setup of Arch and Arch-based Linux systems. With a TUI, package automation, and environment configurations, Carch helps users configure their systems efficiently.

Features

  • Quick Setup: Automates the installation of essential packages for a smooth start.

  • Enhanced TUI Navigation: A color-themed TUI for intuitive navigation and setup.

  • Preconfigured Scripts: Tailored setups for popular window managers and tools, with options like Dwm, Hyprland, i3wm, and more.

  • Continuous Updates: Regular updates and improvements driven by user feedback.

Script Overview

Carch comes with the following scripts to configure various aspects of your Arch-based system:

  • Dwm Setup: Setup Dwm With One Click Script.
  • Hyprland Setup: Sets up the Hyprland compositor with my dotfiles.
  • i3wm Setup: Installs and configures i3 window manager for a lightweight setup.
  • Install Fonts: Install Nerd Fonts.
  • Install LTS Kernel: Installs the LTS kernel.
  • Install Packages: Installs a curated list of essential packages for a functional environment.
  • Setup Alacritty: Configures Alacritty terminal with my configurations.
  • Setup Aur: Installs and configures AUR helpers for easy package management.
  • Setup Fastfetch: Configures Fastfetch for quick system information display.
  • Setup GRUB: Customizes GRUB bootloader with enhanced appearance and settings.
  • Setup Kitty: Sets up Kitty terminal with advanced features.
  • Setup Neovim: Configures Neovim for an optimized code editing experience.
  • Setup Picom: Installs and configures Picom for window compositing and effects.
  • Setup Rofi: Sets up Rofi as an application launcher and switcher.
  • Setup SDDM: Configures the SDDM display manager for a streamlined login experience.
  • Setup Themes-Icons: Installs and applies a selection of themes and icons.
  • SwayWM Setup: Configures Sway window manager for Wayland environments.
  • Wallpapers: Sets up a collection of wallpapers for personalized aesthetics.

Installation

To install Carch, run the following command in your terminal:

bash <(curl -L https://chalisehari.com.np/carch)

After installation, launch Carch by running:

carch

Note: Carch is designed for Ar>ch and Arch-based Linux systems.

Usage

Once installed, launch Carch with carch in your terminal.

Tip: You only need to install Carch once; simply type carch to access it anytime.
