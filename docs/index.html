Introduction

Welcome to the Carch Documentation! Carch is a modular Bash script crafted to streamline the initial setup of Arch and Arch-based Linux systems. It provides an intuitive TUI for package automation and environment configurations, enabling users to set up their systems quickly and easily.

Features

Carch offers a wide range of features to enhance your Linux setup experience:

  • Quick Setup: Easily install essential packages to get your system up and running.
  • Enhanced TUI Navigation: Enjoy a visually appealing TUI with color-coded navigation and setup guidance.
  • Preconfigured Scripts: Get support for popular environments like Dwm and Hyprland, simplifying your configuration process.
  • Continuous Updates: Regular updates are provided based on community feedback, ensuring Carch stays relevant and useful.

Installation

To install Carch, open your terminal and run the following command:

bash <(curl -L https://chalisehari.com.np/carch)

Once installed, launch Carch with the command:

carch

Note: Carch is designed exclusively for Arch and Arch-based Linux distributions.

Usage

After installation, you can start using Carch by typing carch in your terminal. Follow the intuitive TUI steps to configure your environment.

Tip: Carch only needs to be installed once; you can access it anytime with the carch command.
